# Software

On this page I keep a list of Software products that I am (or was) enthusiast about.

## Moving away from Microsoft Windows

After using Microsoft DOS and Windows 95, 98, NT, 2000, Vista, 7, 8, 8.1, 10, 11 since childhood I began switching to Linux in 2025.

(And Microsoft Windows Server 2003, 2008, 2008 R2, 2012, 2012 R2, 2016, 2019, 2022 mostly as ASP.NET Webservers for my former webhosting business.)

**Why?** Microsoft deciding to end Windows 10 and not giving people (with somewhat older but good working PC's) an opportunity to upgrade to Windows 11, and advising to recycle millions of good working PC's!

We have to care about our planet, and big companies should give good examples.

## Linux Zorin OS

So I switched to Linux, I tried Zorin OS and Ubuntu + Kubuntu but I like Zorin OS the most. It's easy to use and has the look and feel of Windows so I feel at home.

You can download and install it on your own PC for free, give it a try: [zorin.com](https://zorin.com/)
