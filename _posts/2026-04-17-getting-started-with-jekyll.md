# Getting Started with Jekyll

Jekyll is a static site generator that transforms your plain text into beautiful static websites.

## Why Jekyll?

1. **Simple** - Write in Markdown
2. **Fast** - No database needed
3. **Secure** - Static files only
4. **Free** - GitHub Pages integration

## How to begin?

1. Create a Github account
2. Create a Repository for your site
3. Go to your Repository Settings > Pages
4. Point your domainname DNS settings to Github
5. Ask your favorite AI tool for further Jekyll intructions
